Code-Sample1:
Code-Sample1 is a 2D game developed using C++ and SDL.
To learn about the game watch InstinctGameVideo explaining rules, controls, scoring and levels of game.
If you have difficulties in playing the video watch on youtube(https://www.youtube.com/watch?v=ODy-7AAq7Qk&feature=youtu.be)
Instinct.pdf is the game document which has detail information about the game. Technical details are mentioned in the document.
To play the game open InstinctGame folder and run SDLgames.exe application.
All the source code is in SourceCode folder.
